# 🦸 Shadow Titans

Um jogo 2D desenvolvido em **HTML5 Canvas**, **CSS** e **JavaScript**, inspirado no universo dos **Jovens Titãs**, com uma campanha dividida em fases, cutscenes, batalhas contra vilões e uma história totalmente original.

---

# 📖 Sobre o projeto

**Shadow Titans** é um jogo de ação 2D onde o jogador assume o controle de um dos membros da equipe **Shadow Titans** para enfrentar uma série de vilões que ameaçam destruir a cidade e dominar a Torre dos Titãs.

Durante a campanha, o jogador acompanha uma narrativa através de cutscenes, enfrenta inimigos em diferentes cenários e avança por diversas fases até a batalha final.

O projeto foi desenvolvido utilizando apenas tecnologias web, explorando os recursos do **HTML5 Canvas** para renderização do jogo.

# 🎮 Como jogar

1. Faça o download ou clone o repositório:

```bash
git clone https://github.com/pedrok9-dev/shadow_titans.git
```

2. Abra a pasta do projeto.

3. Execute o arquivo **index.html** em seu navegador.

Ou utilize uma extensão como **Live Server** no **Visual Studio Code** para uma melhor experiência.

---

# 🎯 Objetivo

Seu objetivo é derrotar todos os vilões presentes nas fases do jogo, acompanhar a história dos **Shadow Titans** e impedir que os inimigos dominem a cidade e a Torre dos Titãs.
